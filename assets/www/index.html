জ্যামিতির সাধারন কতিপয় সুত্র

  • => কার্তেসীয় স্থানাংক কে পোলারে রুপান্তরের সুত্রঃ

          
          

  • => পোলার হতে কার্তেসীয় পদ্ধতিতে রুপান্তরের সুত্রঃ

          
          

  • => P(x 1, y1) এবং Q(x 2, y2) বিন্ধুদয়ের দূরত্বঃ

          

  • => (a) (x 1, y1) এবং (x 2, y2) বিন্দু দুইটির সংযোগ রেখাংশকে ( x,y) বিন্ধুটি m 1 : m 2 অনুপাতে অন্তভিভক্ত করলে

          

        (b) আবার (x,y) বিন্দুটি সংযোগ রেখাংশকে m 1 : m 2 অনুপাতে বহিবিভক্ত করলে

          

        (c) (x 1, y1) এবং (x 2, y2) বিন্দুদ্বয়ের সংযোগ রেখাংশের মধ্যবিন্ধুর স্থানাংক

          

  • => (a) (x 1, y1) , (x 2, y2) এবং (x 3, y3) বিন্দুত্রয় দ্বারা গঠিত ত্রিভুজের ক্ষেত্রফলঃ

          

        (b) উপরোক্ত ত্রিভুজের ভরকেন্দ্রের স্থানাংকঃ

সরলরেখা

  • (a) X অক্ষের সমান্তরাল সরলরেখার সমীকরন, y = b
  • (b) Y অক্ষের সমান্তরাল সরলরেখার সমীকরন, x = a
  • (c) মূলবিন্দুগামী যে কোন সরলরেখার সমীকরন , y = mx . সেখানে রেখার ঢাল m
  • (d) মূলবিন্দুগামী ও (x 1, y1) বিন্দুগামী সরলরেখার সমীকরন,

          
  • (e) Y অক্ষকে নির্দিষ্ট দূরত্বে ছেদ করে এরূপ রেখার সমীকরন, y = mx + c . এটিকে সরলরেখার ঢাল ও বলা হয় ।
  • (f) অক্ষদয়কে ছেদ করে এরূপ রেখার সমীকরনঃ

          
  • (g) মূলবিন্দু থেকে কোন রেখার উপর লম্ব দূরত্ব = p এবং উক্ত লম্বটি X অক্ষের সাথে α কোণ উৎপন্ন করে এরূপ রেখার সমীকরন, x cos α + y sin α = p
  • (h) একটি বিন্দু (x 1, y1) দিয়ে অতিক্রমকারী রেখার সমীকরন,
        y - y 1 = m(x - x 1), যেখানে রেখার ঢাল m
  • (i) (x 1, y1) ও (x 2, y2) বিন্দুগামী যে কোন রেখার সমীকরন,

          
  • (j) (x , y) ও (x 1, y1) বিন্দুদ্বয়ের দূরত্ব = r এবং বিন্দুদ্বয়ের সংযোগ রেখাংশ X অক্ষের সাথে θ কোণ উৎপন্ন করলে রেখাটির সমীকরনঃ

          
  • (k) (x 1, y1) ও (x 2, y2) বিন্দুদ্বয়ের সংযোগ রেখাংশের ঢালঃ

          
  • (l) y = m 1x + c 1 ও y = m 2x + c 2 রেখাদ্বয়ের মধ্যবর্তী কোণ θ হলেঃ

          
       # এ রেখা দুইটি সমান্তরাল হলে , m 1 = m 2
       # এরা পরস্পর লম্ব হলে, m 1 X m 2 = -1
  • (m) ax + by + c = 0 রেখার সমান্তরাল এরূপ যেকোন রেখার সমীকরন
        ax + by + k = 0, যেখানে k একটি ইচ্ছামূলক ধ্রবক।
  • (n) ax + by + c = 0 রেখার ওপর লম্ব এরূপ যেকোন রেখার সমীকরন
        bx - ay + k = 0, যেখানে k একটি ইচ্ছামূলক দ্রুবক।
  • (o) a 1x + b1y + c1 = 0 এবং a2x + b2y + c 2 = 0 রেখাদ্বয়ের ছেদবিন্দু গামী যে কোন সরলরেখার সমীকরন a1x + b1y + c1 + k (a 2x + b 2y + c 2) = 0 , যখন k একটি অশূন্য ধ্রবক ।
  • (p) a 1x + b1y + c1 = 0 , a2x + b2y + c 2 = 0 এবং a 3x + b3y + c3 = 0 রেখা ত্রয় সমবিন্দু হলে

          
  • (q) ax + by + c 1 = 0 , ax + by + c 2 = 0 সমান্তরাল রেখাদ্বয়ের মধ্যবর্তী দূরত্বঃ

          
  • (r) (x 1 , y 1) বিন্দু থকে ax + by + c = 0 রেখার উপর অঙ্কিত লম্ব - দূরত্বঃ

          
  • (s) a 1x + b1y + c1 = 0 এবং a2x + b2y + c 2 = 0
    • রেখাদ্বয়ের অন্তর্ভূক্ত সমদ্বিখন্ডকের সমীকরনঃ

কনিক সংক্রান্ত সুত্র

  • (1) পরাবৃত্ত এর মান সমীকরনঃ y 2 = 4ax

        শীর্ষ বিন্দুর স্থানাংকঃ ( 0,0)
        উপকেন্দের স্থানাংকঃ (a,0)
        দিকাক্ষের সমীকরণঃ x = -a
        অক্ষরেখার সমীকরনঃ y = 0
        উপকেন্দ্রিক লম্বের দৈর্ঘ = 4a
        উপকেন্দ্রিক লম্বের সমীকরনঃ x = a

  • (2) অধিবৃত্ত এর মান সমীকরনঃ

          

        কেন্দ্রের স্থানাংকঃ (0,0)
        উপকেন্দ্র দুটির স্থানাংকঃ (±ae, 0)
        শীর্ষবিন্দুর স্থানাংকঃ (±a, 0)
        আড় অক্ষের সমীকরণঃ y = 0
        অনুবন্ধি অক্ষের সমীকরণঃ x = 0

        দিকাক্ষের সমীকরণঃ

       উপকেন্দ্রিকতাঃ

        উপকেন্দ্রিক লম্বঃ

        উপকেন্দ্রিক লম্বের সমীকরনঃ x = ±ae
        অক্ষ দুইটির দৈর্ঘঃ 2a ও 2b

  • (3) উপবৃত্ত এর মান সমীকরনঃ

          

        কেন্দ্রের স্থানাংকঃ (0,0)
        বৃহৎ অক্ষ = 2a
        ক্ষদ্র অক্ষ = 2b
        উপকেন্দ্র এর স্থানাংকঃ (±ae, 0)
        শীর্ষবিন্দুর স্থানাংকঃ (±a, 0)
        বৃহৎ অক্ষের সমীকরণঃ y = 0
        ক্ষুদ্র অক্ষের সমীকরণঃ x = 0

        দিকাক্ষের সমীকরণঃ

       উপকেন্দ্রিকতাঃ

        উপকেন্দ্রিক লম্বঃ

        উপকেন্দ্রিক লম্বের সমীকরনঃ x = ±ae

বৃত্ত সংক্রান্ত সুত্র

  • (1) কেন্দ্র (0,0) এবং r ব্যাসার্ধবিশিষ্ট বৃত্তের সমীকরনঃ x 2 + y 2 = r 2
  • (2) কেন্দ্র (h,k) এবং r ব্যাসার্ধবিশিষ্ট বৃত্তের সমীকরনঃ

          
  • (3) বৃত্তের সাধারন সমীকরনঃ x 2 + y 2 + 2gx + 2fy + c = 0
        # যার কেন্দ্রঃ (-g, -f)

       # এবং ব্যাসার্ধঃ

  • (4) (x 1, y1) ও (x 2, y2) বিন্দু দুটিকে ব্যাস ধরে অঙ্কিত বৃত্তের সমীকরনঃ

          

  • (5) কোন বৃত্তের উপরিস্ত (x 1, y 1) বিন্দুতে অঙ্কিত স্পর্শকের সমীকরণঃ

          
       ( যখন বৃত্তের সমীকরণঃ x 2 + y 2 = a 2 )

  • (6) কোন বৃত্তের উপরিস্ত (x 1, y 1) বিন্দুতে অঙ্কিত স্পর্শকের সমীকরণঃ

          
       ( যখন বৃত্তের সমীকরণঃ x 2 + y 2 + 2gx + 2fy + c = 0 )
